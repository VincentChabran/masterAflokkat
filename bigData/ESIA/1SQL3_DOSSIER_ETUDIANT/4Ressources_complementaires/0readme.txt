Pour réaliser les différents exercices du MOOC 
MOOCEBFR3 : Ingénierie des Données du Big Data: SGBD Objet relationnel et SQL3 Oracle 
Un certain nombre d'outils et environnements sont nécessaires :
- Le SGBD Oracle
- Le client Sqldeveloper Oracle
- Le client sqlplus dans instant client Oracle
- Le jdbk java Oracle
- L'IDE intellij
- Dev C++

Vous devez procéder à l'installation de ses outils et environnements.

1) Le dossier : 1Installation_Deinstallation_Configuration_ORACLE

Ce dossier contient la procédure d'installation, déinstallation et configuration du SGBD Oracle 21c.
Vous aurez avec cette installation l'environnement Oracle dans votre
poste de travail.
Cet environnement pourra être utilisée dans tous les exercices de ce Mooc.

Plus particulièrement si vous avez besoin d'accéder système de fichier de la machine ou le SGBD tourne.
Par exemple pour créer une Object Directory.
Par exemple les exercices des modules 3.5 et 3.6a

2) Le dossier : 2Installation_Configuration_Sqldeveloper

Ce dossier contient la procédure d'installation de sqldeveloper Oracle.
Il s'agit d'un environnement graphique pour interargir avec le SGBD Oracle.

Cet environnement pourra être utilisée dans tous les exercices de ce Mooc.

3) Le dossier : 3Installation_Configuration_instantClient

Ce dossier contient la procédure d'installation de instant Oracle.
Instant client fournit un ensemble d'outils clients pour interargir avec le SGBD Oracle.

Un de ses clients est sqlplus.
Vous pouvez utiliser indifférement sqldeveloper ou sqlplus.
Sqlplus a l'avantage d'afficher les pointeurs REF.

Cet environnement pourra être utilisée dans tous les exercices de ce Mooc.

4) Le dossier : 4Installation_Jdk_Java

Ce dossier contient la procédure d'installation du jdk java Oracle.
Il sera nécessaire pour compiler les classes java.

Concerne les exercices module 3.6

5) Le dossier : 5Installation_Ide_Inellij

Ce dossier contient la procédure d'installation de l'IDE intellij.
Il sera nécessaire pour compiler les classes java.

Exercices module 3.6

5) Le dossier : 6DevCpp

Ce dossier contient la procédure d'installation de DEV C++.
Il sera nécessaire pour construire une DLL C++.

Concerne les exercices du module 3.6 pour appeler depuis Oracle des fonctions définies dans une librairie C/C++.

Utilisateur uniquement avec le SGBD installé en local sur votre poste.
En effet, vous avez besoin d'accéder système de fichier de la machine ou le SGBD tourne.
