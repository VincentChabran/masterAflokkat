Vous trouverez dans ce dossier les installations nécessaires pour les exercices du 
cours Big Data.

Pour effectuer les exercices autour :
	- Oracle NOSQL vous devez utiliser les installations MV BIGDATA dossier 3... 	(recommandé). Les installations du dossier 1.. sont bonnes pour une prise 	main d'Oracle NOSQL en stand alone
	 - MongoDB vous devez utliser les installations du du dossier 4. Vous pouvez aussi utilser les installations de la MV Big Data dossier 3.. MongoDB y est installé

	- Construction de lacs de données : vous devez utiliser en priorité la MV Big Data dossier 3. Il est aussi possible d'utiliser les installations de MV Big Data Oracle (attention les composants sont un peu vieux)
